# Twitter Monitor Plugin

将原始的 twitter_monitor 脚本改造成 Open-AwA 标准插件后，这个插件可以直接被平台发现、加载，并以工具形式提供给 AI 调用。

主要能力：

- 抓取一个或多个 Twitter 账号的最新推文
- 缓存最近抓取结果与按日累积结果
- 查询缓存统计、搜索用户、读取账号信息
- 整理适合当前对话模型直接总结的推文素材

配置方式：

- 在 config.json 或插件配置页中填写 twitter_api_key
- monitored_users 支持用逗号分隔多个默认监控对象

运行说明：

- 插件首次执行前，需要为它授权所需的网络和文件权限
- 抓取结果保存在当前插件目录下的 data 目录中
- 插件不会再调用独立的总结模型；需要总结时，直接让当前对话模型读取插件返回的数据并完成总结
