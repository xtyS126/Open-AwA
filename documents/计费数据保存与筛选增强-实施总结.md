## 使用说明

### 查看全部数据
1. 访问计费页面：http://localhost:5173/billing
2. 在时间筛选下拉框中选择"全部"
3. 即可查看所有历史计费数据

### 配置数据保留
1. 访问设置页面：http://localhost:5173/settings
2. 点击"数据保留"标签页
3. 设置最大保存天数（默认365天）
4. 可选：勾选"保存后清理超出保留期限的旧数据"
5. 点击"保存设置"

### API调用示例

#### 获取保留配置
```bash
curl http://localhost:8000/api/billing/retention
```

返回示例：
```json
{
  "retention_days": 365,
  "total_records": 1234,
  "oldest_record": "2024-01-01T00:00:00",
  "newest_record": "2026-03-22T10:30:00"
}
```

#### 更新保留配置
```bash
curl -X PUT "http://localhost:8000/api/billing/retention?retention_days=180&cleanup=true"
```

返回示例：
```json
{
  "success": true,
  "old_retention_days": 365,
  "new_retention_days": 180,
  "deleted_records": 45
}
```
