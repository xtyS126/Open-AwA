# 插件导航与配置面板性能验收记录（2026-04-12）

## 1. 验收范围

本记录对应 `.trae/specs/migrate-plugin-nav-and-config-pages` 中 Task 6 的性能验收子项，关注以下指标：

- 插件包（<=50MB）导入耗时 `< 3s`
- 配置表单首渲染耗时 `< 300ms`
- 连续快照内存增长 `<= 1MB`

## 2. 验收环境

- 操作系统：Windows 11
- 前端：`frontend/`（React + Vite + Vitest）
- 验收日期：2026-04-12
- 测试命令：`npm run test:coverage`

## 3. 验收方法

### 3.1 导入耗时（本地 ZIP 与 URL 导入路径）

- 使用 `PluginsPage` 单元测试覆盖导入解析逻辑：
  - ZIP 扩展名 / MIME / 大小校验（含 50MB 上限约束）
  - URL 去空白后导入
  - 导入成功后的刷新与提示反馈
- 观察同轮执行日志中 `PluginsPage` 测试文件耗时。

### 3.2 配置表单首渲染

- 使用 `PluginConfigPage` 单元测试覆盖 schema 拉取与五类控件渲染路径。
- 观察同轮执行日志中 `PluginConfigPage` 测试文件耗时。

### 3.3 内存增长

- 在同一轮前端测试执行期间，连续运行并对比进程快照，记录前后差值。
- 该指标用于验证本次改动未引入明显前端内存异常增长。
