# 终端错误修复总结

## 问题诊断

### 1. 后端端口冲突
**错误信息**：
```
ERROR: [Errno 10048] error while attempting to bind on address ('0.0.0.0', 8000): 
[winerror 10048] 通常每个套接字地址(协议/网络地址/端口)只允许使用一次。
```

**原因**：端口8000被之前的进程占用

**解决方案**：
```bash
# 查找占用端口的进程
netstat -ano | findstr ":8000"

# 终止该进程
taskkill /F /PID <进程ID>
```

**结果**：成功释放端口并重启后端服务
