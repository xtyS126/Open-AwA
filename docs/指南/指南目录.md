# 使用与部署指南

本目录存放开发、测试、部署与迁移相关的操作性文档，适合作为环境搭建和交付阶段的入口。

## 文档列表

- [deployment.md](deployment.md)：本地开发、环境变量与部署说明
- [deployment-migration-guide.md](deployment-migration-guide.md)：部署迁移与兼容处理说明
- [testing.md](testing.md)：后端、前端与 E2E 测试策略

## 使用建议

- 本地启动项目或新环境部署时，从 [deployment.md](deployment.md) 开始。
- 版本迁移或环境切换时，优先查看 [deployment-migration-guide.md](deployment-migration-guide.md)。
