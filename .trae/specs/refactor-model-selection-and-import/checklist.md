# Checklist

## 后端接口与数据管理
- [x] 获取模型列表接口不再意外覆盖本地用户的精细配置。
- [x] 支持通过接口批量导入指定的模型。
- [x] 支持通过接口批量删除指定的已导入模型。

## 前端配置页交互
- [x] 点击“获取模型列表”有明确的 Loading 状态。
- [x] 获取成功后弹出对话框，列表展示所有可用模型供用户多选。
- [x] 导入成功后，配置页区域仅显示已导入的模型。
- [x] 支持在已导入列表中勾选模型，并使用“批量删除”按钮移除。
- [x] 删除操作有二次确认弹窗。

## 状态同步与聊天页
- [x] 聊天页的模型下拉选择框（`select`）仅展示当前已导入的模型。
- [x] 在设置页导入或删除模型后，聊天页下拉框状态能实时同步更新。

## 异常与体验
- [x] 各种网络请求失败时（如获取失败、导入失败）均有友好的 Toast/Message 提示。
- [x] 验证通过：模型列表获取、导入、删除、状态同步等核心流程在多种网络条件下均能稳定运行。
