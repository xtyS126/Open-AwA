# Checklist

## 范围与目标
- [x] 仅新增“删除供应商”能力，不扩展无关功能
- [x] 删除能力覆盖前端交互、接口调用、后端处理三层

## 前端交互
- [x] 右侧详情区出现“删除供应商”按钮
- [x] 删除前有明确确认步骤
- [x] 删除成功后列表刷新并自动切换到可用项或空状态
- [x] 删除失败后保留当前表单内容并提示错误

## 后端能力
- [x] 提供 `DELETE /billing/providers/{provider}` 接口
- [x] 存在供应商时可删除并返回成功
- [x] 不存在供应商时返回 404
- [x] 删除逻辑与当前软删除策略一致（is_active=false）
