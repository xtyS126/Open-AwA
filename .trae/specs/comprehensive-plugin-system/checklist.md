# Checklist

## 功能完整性
- [x] 生命周期状态机可用，状态迁移受控且幂等
- [x] 生命周期失败可自动回滚到稳定状态
- [x] 支持本地 ZIP、远程 URL、npm 三类来源加载
- [x] 每个插件运行于独立沙箱并具备资源限制
- [x] 至少 8 类扩展点可注册、可执行、可校验
- [x] manifest 中扩展点与版本范围校验通过
- [x] 依赖图可构建，冲突可检测并返回建议
- [x] 危险操作触发权限申请并可撤销
- [x] 热更新采用双缓存策略并支持失败回滚
- [x] 灰度发布支持按用户、地域、版本分批策略

## 安全与可观测性
- [x] 静态扫描可阻断危险代码模式
- [x] 运行时拦截可阻断敏感 API 访问
- [x] 每个插件生成独立日志并支持 DEBUG 动态开关
- [x] 调试面板可查看 API 调用、事件、性能指标

## 工具链与交付
- [x] 官方 CLI 支持 init/dev/typecheck/build/sign/publish
- [x] 插件包命名与目录结构符合规范
- [x] 《插件开发手册》与 SDK 类型声明完整可用
- [x] 三个官方示例插件可安装、可运行、可调试

## 验收标准
- [ ] 安装 50 个插件后首屏渲染增加 < 100ms
- [ ] 安装 50 个插件后内存增长 < 30MB
- [ ] 插件安装成功率 ≥ 99.5%
- [ ] 热更新失败自动回滚且用户无明显中断
- [ ] 第三方安全审计高危漏洞为零
- [ ] Chrome/Edge/Firefox/Electron 最新稳定版 E2E 全通过
