# Checklist

## API URL 保存与格式校验
- [x] 系统能验证 URL 格式是否合法，不合法时给出错误提示
- [x] 输入如 `https://api.deepseek.com/`（无 v1）时，保存后自动变为 `https://api.deepseek.com/v1`
- [x] 输入如 `https://api.deepseek.com/v1`（有 v1）时，保存后保持原样

- [x] 系统在获取模型列表时，自动向基础 URL 添加 `/chat/completions` 后缀发起请求
- [x] 获取模型列表请求失败时，有清晰的异常捕获与错误提示
- [x] 有明确的日志记录发起模型列表请求的完整 URL 与结果

- [x] 有明确的日志记录聊天请求的完整 URL 与结果
